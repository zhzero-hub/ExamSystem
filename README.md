# ssm-Online_Examination
南京航空航天大学本科软件工程课程设计
在线考试系统
2020年秋

### 简介

> 在线考试系统分为3种角色，admin，teacher，student
>
> - admin负责整个系统的操作，包含题库管理，知识点管理，教师管理，考试管理，基础信息管理
> - teacher负责题库管理，考试管理，学生管理
> - student是系统的主要使用用户，包含考试系统，试卷回顾


### 技术栈

- Spring+SpringMvc+MyBatis
- mysql
- bootstrap等
